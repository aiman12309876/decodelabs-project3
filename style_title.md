# Style Tile - DecodeLabs UI/UX Project 3

## Brand Overview
- **Brand Name:** FreshBite
- **Industry:** Premium Organic Food Delivery
- **Vibe:** Modern, Fresh, Trustworthy, Healthy, Fast
- **Target Audience:** Health-conscious professionals, age 25-45

## 1. Color Palette

### Primary Colors
| Color | Hex Code | Usage |
|-------|----------|-------|
| Fresh Green | #2ECC71 | Primary CTA, Brand Accent |
| Dark Green | #1A7A3A | Headings, Dark Elements |
| White | #FFFFFF | Background, Text |

### Secondary Colors
| Color | Hex Code | Usage |
|-------|----------|-------|
| Mint Green | #A8E6CF | Background Highlights |
| Soft Yellow | #F9E79F | Accent, Badges |
| Light Gray | #F5F5F5 | Section Backgrounds |

### Color Psychology
| Color | Meaning |
|-------|---------|
| Green | Health, Freshness, Nature, Organic |
| White | Clean, Pure, Trust |
| Yellow | Energy, Warmth, Appetite |

## 2. Typography

### Primary Font: Poppins
| Style | Font Size | Weight | Usage |
|-------|-----------|--------|-------|
| H1 | 32px | Bold (700) | Main Headings |
| H2 | 24px | Semi-Bold (600) | Section Titles |
| H3 | 18px | Medium (500) | Card Titles |
| Body | 16px | Regular (400) | Paragraphs |
| Small | 14px | Regular (400) | Subtext, Labels |

### Font Characteristics
- **Poppins:** Modern, Clean, Geometric, Highly Readable
- **Google Font Link:** `https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700&display=swap`

## 3. UI Elements

### Buttons
| Type | Background | Text | Hover |
|------|------------|------|-------|
| Primary | #2ECC71 | White | Darken to #27AE60 |
| Secondary | Transparent | #2ECC71 | Background #2ECC71, White Text |

### Cards
- Background: White (#FFFFFF)
- Border Radius: 16px
- Box Shadow: `0 4px 15px rgba(0,0,0,0.08)`
- Padding: 20px

### Input Fields
- Border: 1px solid #E0E0E0
- Border Radius: 8px
- Focus: Border #2ECC71

---

## 5. Design Principles

| Principle | Application |
|-----------|-------------|
| Visual Hierarchy | Green for CTAs, larger text for headlines |
| Consistency | Same colors, fonts, spacing across all screens |
| Whitespace | Generous padding for clean look |
| Accessibility | High contrast for readability |
| Visual Appeal | High-quality food imagery |
