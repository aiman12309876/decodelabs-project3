# ATM Interface - DecodeLabs Java Project 3

## What It Does
A Java-based ATM simulation using Object-Oriented Programming principles.

## Features
- PIN validation
- Check balance
- Deposit money
- Withdraw money
- Account information display
- Input validation

## Classes

### BankAccount
| Attribute | Description |
|-----------|-------------|
| accountNumber | Unique account ID |
| accountHolder | Account owner name |
| balance | Current balance |
| pin | Security PIN |

### ATM
| Method | Description |
|--------|-------------|
| displayMenu() | Show menu options |
| run() | Main ATM loop |
| main() | Entry point |

## How to Run
1. Compile: `javac BankAccount.java ATM.java`
2. Run: `java ATM`
3. Enter PIN: `1234`
